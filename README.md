# Personal Portfolio Website

A clean and responsive personal portfolio website built using **HTML and CSS**.  
This project showcases my profile, skills, projects, and contact details in a
simple and professional layout.

### Features
- Responsive design
- Clean and modern UI
- About, Skills, Projects, Resume, and Contact sections
- Beginner-friendly and easy to customize

### Technologies Used
- HTML5
- CSS3

This project helped me strengthen my fundamentals of web development and
understand proper website structuring and styling.
